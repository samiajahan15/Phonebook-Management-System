# Phonebook-Management-System
A C++ phonebook management system using automatic sorting and binary search

## Project Overview
This project is a menu-driven phonebook management system developed in C++. It stores contacts, automatically sorts them by name using Bubble Sort, and performs fast searching using Binary Search.

## Features
- Add Contact
- Display Contacts
- Search Contact
- Edit Number
- Delete Contact
- Automatic Sorting by Name
- Edge Case Handling

## Algorithms Used
- Bubble Sort
- Binary Search

## Data Structure
- `struct Contact`
- `vector<Contact>`

## Edge Cases Handled
- Empty phonebook
- Empty input
- Duplicate names
- Contact not found
- Invalid menu choice

## How to Run
Compile the code using a C++ compiler and run the executable.
